## minder repo

Manage repositories

### Synopsis

The repo commands allow the management of repositories within Minder.

```
minder repo [flags]
```

### Options

```
  -h, --help              help for repo
  -j, --project string    ID of the project
  -p, --provider string   Name of the provider, i.e. github
```
